# TerraRoute

## Know the road before you drive.

TerraRoute is an AI-powered Road Intelligence Platform that recommends the safest and most suitable routes based on:

- Vehicle Compatibility
- Road Quality
- Weather
- Terrain
- Route Intelligence
- AI Recommendations

### Tech Stack

#### Frontend
- React
- Material UI
- Leaflet

#### Backend
- Python
- FastAPI
- SQLAlchemy

#### Database
- MySQL
